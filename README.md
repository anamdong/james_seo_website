# James Seo Portfolio (Static)

This is a pure static site (HTML/CSS/JS), so it works directly on GitHub Pages with no build step.

## Files

- `index.html`: page structure and content
- `styles.css`: layout, color system, typography, and hover effects
- `script.js`: year update + reveal animation for project cards

## Customize quickly

- Update menu/sections in `index.html`
- Replace project card text and image URLs in `index.html`
- Change colors in `styles.css` under `:root`
- Update contact links in the footer

## Deploy on GitHub Pages
